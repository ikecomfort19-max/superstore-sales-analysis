# Superstore Sales Analysis

## Project Overview
SQL analysis of a retail superstore dataset containing 9,800+ orders.

## Business Questions Answered
1. What is the total revenue generated?
2. Which region generates the most sales?
3. Which product category is most popular?
4. Who are the top 5 customers by sales?
5. Which ship mode is used most often?
6. Are there orders where the company made a loss?
7. Which sub-categories perform above category average?
8. Who are the top 2 customers per region?
9. How does each state rank within its region?
10. What is the running total of sales per category?

## Tools Used
- MySQL Workbench
- Dataset: Superstore Sales Dataset (Kaggle)
